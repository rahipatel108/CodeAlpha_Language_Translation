# 🌍 Language Translation Tool

A simple Language Translation Tool built using Python and Streamlit. This application automatically detects the input language and translates it into the selected target language.

## Features
- Automatic language detection
- Translate text into multiple languages
- Simple and user-friendly interface
- Built with Streamlit

## Technologies Used
- Python
- Streamlit
- deep-translator

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/rahipatel108/CodeAlpha_Language_Translation.git
```

2. Install dependencies:
```bash
pip install streamlit deep-translator
```

3. Run the application:
```bash
streamlit run app.py
```
